# 17Tech Redesign

Static marketing site redesign for 17Tech.

## Local preview

Open `index.html` directly, or run a static server from this folder.

## Deploy to Vercel

1. Push this folder to a GitHub repository.
2. Import the repository into Vercel.
3. Use the project root as-is; no build command is required.
4. Leave the output directory empty.

## Files

- `index.html`
- `style.css`
- `script.js`
- `assets/`
